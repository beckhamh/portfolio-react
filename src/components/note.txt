page web projet 1:

site web developpé en Node.js durant mon parcour scolaire, les utilisateurs peuvent se crée un compte ensuite pouvoir selectionner des items 
(ajouter, supprimer, modifier) et les mettre dans le panier et soumettre sa commande.

page d'inscription: un courriel valid et un mot de passe pour pouvoir s'inscrire et crée un compte

les utilisateurs qui possèdent un compte peuvent se connecter facilement en utilisant son adresse mail et le mot de passe

selectionner un item en cliquand sur le bouton ajouter au panier, cela vas mettre le produit selectionner automatiquement dans votre panier

tous les produits selectioner se trouveront ici dans la panier, pour supprimer un item faut just cliquer sur (x), pour valider la commande on doit cliquer sur le bouton soummetre

la page commande n'est pas accecible a tout le monde, seule l'administrateur qui a acces pour voir les commandes qui sont soumisent et pouvoir changer le status de chacune
